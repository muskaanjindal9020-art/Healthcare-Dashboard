# Healthcare Provider Dashboard (Power BI)

## 📌 Overview
A Power BI dashboard analyzing billing, insurance coverage, and diagnosis-level cost patterns for a healthcare provider. Built to practice DAX, data modeling, and interactive report design.

## 🎯 Business Problem
 The provider wanted visibility into which procedures and diagnoses drive the highest billing amounts, and how much of that cost is covered by insurance vs. paid out-of-pocket by patients.

## 📊 Data Source
Public dataset from youtube
Tables used: patients, procedures, diagnoses, insurance, departments


## 🛠️ Tools & Skills Used
- Power BI (data modeling, DAX measures, interactive visuals)
- Power Query (data cleaning/transformation)
- DAX for calculated KPIs (Total Billing, Insurance Coverage %, Revenue per Patient)

## 🔑 Key Features
- KPI cards: Total Billing Amount, Insurance Coverage %, Out-of-Pocket %, Revenue per Patient
- Billing breakdown by Procedure (X-Ray, CT Scan, MRI, Ultrasound, Blood Test)
- Billing by Diagnosis × Service Type (Emergency / Inpatient / Outpatient)
- Interactive City/Region filters affecting all visuals

## 💡 Key Insights
-X-Ray accounts for 31% of total billing, the single largest procedure cost driver.
- Hypertension has the highest outpatient billing among all diagnoses, at $0.72M.
- Out-of-pocket costs remain roughly one-third of total billing across most diagnoses.
